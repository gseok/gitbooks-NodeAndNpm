#### npm으로 npm 버전 업데이트 하기

#### 

npm은 node module을 관리해주는 package manager이다. \(정확하게는 javascript library을 관리해주는\)

npm은 npm자신의 버전을 update 해주는 기능을 가지고 있다.



##### npm 버전확인

```bash
$ npm --version
```



##### npm 버전 update

```bash
$ npm install -g npm
```



이와 같이 간단하게 npm의 버전을 확인하고 update할 수 있다.

가끔 실행해서 가장 최신 버전을 유지하자
